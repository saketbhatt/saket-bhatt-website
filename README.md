# Saket Bhatt - Portfolio Website

A professional portfolio website showcasing my experience, skills, and achievements as a Software Engineer.

## Current Implementation

**Theme:** Professional sci-fi inspired design with dark theme, golden ratio background, and smooth animations.

## Sections

- **Hero:** Introduction with Stephen Hawking quote and call-to-action buttons
- **About:** Personal introduction and philosophy
- **Experience:** Work history at RoZetta Technology, Amazon, and Disney+ Hotstar
- **Highlights:** Awards, learning profiles, key courses, certifications, and academic achievements
- **Skills:** Technical skills organized by categories (Languages, CS Concepts, Tools/Platforms, DevOps)
- **Education:** Academic background from NIT Silchar and Kendriya Vidyalaya
- **Contact:** Links to email, LinkedIn, GitHub, and Topmate

## Features

- **Responsive design** (mobile-first approach with tablet and desktop breakpoints)
- **Dark theme** with gold/orange/teal accent colors
- **Golden ratio background image** with subtle overlay
- **Smooth scrolling navigation** with fixed navbar
- **Hover effects and animations** with sci-fi aesthetic
- **Time-based theme variations** (subtle accent color changes)
- **Intersection Observer** for scroll animations
- **Professional typography** with proper contrast

## Technologies Used

- **HTML5:** Semantic markup with proper meta tags
- **CSS3:** Custom properties, Grid, Flexbox, animations, responsive design
- **JavaScript:** ES6+, Intersection Observer API, smooth scrolling
- **Design:** Sci-fi inspired with professional color scheme and golden ratio aesthetics

## Key Features Implemented

- **Responsive Navigation:** Fixed navbar with smooth scroll links
- **Hero Section:** Full-height with golden ratio background and gradient overlay
- **Interactive Cards:** Hover effects with glowing borders and transforms
- **Smooth Animations:** Fade-in effects on scroll using Intersection Observer
- **Professional Links:** All external links open in new tabs with proper rel attributes
- **Performance Optimized:** Minimal dependencies, optimized images
- **Mobile Responsive:** Comprehensive breakpoints for all device sizes

## How to Run Locally

1. Clone this repository
2. Open `index.html` in your browser (no build step needed)
3. All assets are included and ready to run

## Deploy to GitHub Pages

1. Push all files to your `main` branch
2. Enable GitHub Pages in repository settings
3. Set source to root (`/`)
4. Visit your site at `https://saketbhatt.github.io/saket-bhatt-website/`

## Browser Support

- Modern browsers with CSS Grid and Flexbox support
- Responsive design works on all screen sizes (mobile, tablet, desktop)
- Reduced motion support for accessibility
- WebP image format support for optimized loading
